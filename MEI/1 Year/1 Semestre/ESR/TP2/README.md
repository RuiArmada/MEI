# TP-ESR

A Twitch like application for video streaming

## How to Run

### Client

``` 
chmod + runC.sh
    
./runC.sh
 ```
 
 ### Server
 
 ```
 chmod + runS.sh
    
 ./runS.sh
 ```
 
### Router
 
 ```
 chmod + runR.sh
    
 ./runR.sh
 ```
